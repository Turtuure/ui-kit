# Security Policy

## Reporting a Vulnerability

We take the security of the Archon UI Kit seriously. If you discover a security vulnerability, we appreciate your help in disclosing it to us in a responsible manner.

### How to Report

Please report security vulnerabilities by emailing us at **security@archon.local** (placeholder email).

When reporting, please provide as much detail as possible, including:
*   A clear and concise description of the vulnerability.
*   Steps to reproduce the vulnerability.
*   The affected version(s) of the Archon UI Kit.
*   Any potential impact of the vulnerability.
*   If possible, suggest a fix or mitigation.

### Our Commitment

*   We will acknowledge receipt of your report within 2 business days.
*   We will provide a timeline for addressing the vulnerability if it is confirmed.
*   We will keep you informed of our progress throughout the process.
*   We will publicly acknowledge your responsible disclosure, if you agree, once the vulnerability has been fixed.

Please do not disclose the vulnerability publicly until we have had a reasonable time to address it.

Thank you for helping to keep the Archon UI Kit secure.
